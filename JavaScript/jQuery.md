# jQuery
在以前各種瀏覽器都還擁有許多使用者的時候，
每個瀏覽器都有不同的支援，有時候寫程式可還要考慮不同瀏覽器的支援程度，所以才延生出了 jQuery ，可以只寫 jQuery 就讓不同的瀏覽器都可以支援。

在 jQuery 裡面，"$" 也可以代表 jQuery，

### $('css selector')
選取 DOM Element

### $('documnt').ready(handler)
當整份文件 load 完之後

## Event
### mouse event:
- .click()
- .hover()

### Keyboard event:
- .keydown()

## 控制 CSS 的功能
- .hide() 隱藏
- .show() 顯示
- .fadeOut() 漸變隱藏
- .fadeIn() 漸變顯示
