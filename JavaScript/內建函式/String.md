# String 的內建函式
- String.toUpperCase()
	- 將所有字母轉換成大寫
- String.toLowerCase()
	- 將所有字母轉換成小寫
- String.charCodeAt()
	- 回傳字母的 ASCII 的值
```js
console.log("A".charCodeAt())  // 65
console.log("Z".charCodeAt())  // 90
console.log("a".charCodeAt())  // 97
console.log("z".charCodeAt())  // 122
```
所以可以知道小寫字母介在 97 ~ 122 之間，大寫字母介在 65 ~ 90 之間

- String.fromCharCode()
	- 字母大小寫轉換
	- 大小寫判斷
- String.indexOf()
- String.replace()
	- 可以搭配正規表達式
- String.split()
- String.trim()
- String.length
	- 可以用陣列的方式取出字串的單一字母
