# Data Type 資料型態
## typeof
可以顯示變數的資料型態，回傳的 type 是 string

## Data type 的種類
- Primitive
	- boolean : true , [[false]] , 
	- number : 不管是整數或是小樹都算 number  ( 內建函式可參考 [[Number]] )
	- string : 字串  ( 內建函示可參考 [[String]] )

- undefined : 變數沒有賦予值
- object  #object 
	- 物件 [[Object 物件]]
	- 陣列 [[Array 陣列]]
	- **null**
- function
